# Buy Links Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist `productSearchQuery` from AI recommendations to the database and render a Google Shopping link next to product names in task cards.

**Architecture:** Add a nullable `productSearchQuery` column to `LawnTask`, pass it through from both the analyze and recommendations API routes when creating tasks, then add it to every task SELECT query and render a linked shopping button in `TaskList` and `TaskCard`.

**Tech Stack:** Prisma (PostgreSQL migration), Next.js App Router server components, React/TypeScript client component (`TaskList`).

### Task 4: Render the shopping link in TaskList

**Files:**
- Modify: `components/dashboard/TaskList.tsx` — `Task` interface + `TaskCard` render
- Modify: `components/dashboard/DashboardInteractiveSection.tsx` — `Task` interface

The Google Shopping search URL format is:
`https://www.google.com/search?tbm=shop&q=ENCODED_QUERY`

The link should appear inline next to the product name (as a small icon-only button so it doesn't clutter the card). Use `ShoppingCart` icon from `lucide-react` at `w-3.5 h-3.5`.

- [ ] **Step 1: Write the failing test**

Create `components/dashboard/__tests__/TaskList.test.tsx`:

```typescript
import { render, screen } from "@testing-library/react";
import { describe, it, expect } from "vitest";
import { TaskList } from "../TaskList";

const baseTask = {
  id: "t1",
  title: "Apply fertilizer",
  description: "Apply now",
  priority: "medium",
  status: "pending",
  scheduledStart: null,
  scheduledEnd: null,
  overdueNote: null,
  stillWorthDoing: null,
  product: "Scotts Turf Builder",
  applicationRate: "3 lbs/1000 sq ft",
  spreaderSetting: null,
  taskMode: null,
  productSearchQuery: null,
};

describe("TaskList product shopping link", () => {
  it("renders a shopping link when productSearchQuery is set", () => {
    const task = { ...baseTask, productSearchQuery: "Scotts Turf Builder 32lb" };
    render(<TaskList tasks={[task]} />);
    const link = screen.getByRole("link", { name: /shop/i });
    expect(link).toHaveAttribute(
      "href",
      "https://www.google.com/search?tbm=shop&q=Scotts%20Turf%20Builder%2032lb"
    );
    expect(link).toHaveAttribute("target", "_blank");
  });

  it("does not render a shopping link when productSearchQuery is null", () => {
    render(<TaskList tasks={[baseTask]} />);
    expect(screen.queryByRole("link", { name: /shop/i })).toBeNull();
  });
});
```

    <span className="font-medium">{task.product}</span>

      <a
        href={`https://www.google.com/search?tbm=shop&q=${encodeURIComponent(task.productSearchQuery)}`}
        target="_blank"
        rel="noopener noreferrer"
        aria-label="Shop"
